# better_dictate
A tool for general purpose dictation on a desktop box

## Installation

- `cd better_dictate/env`
- `./create_virtualenv.sh`


## Execution

- In one terminal window:
    - `cd better_dictate`
    - `./run.sh`
- In a second terminal window:
    - `cd better_dictate/frontend`
    - `./run.sh`
